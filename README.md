# hexlet-java-password-generator-bot

Это простой телеграм-бот, позволяющий генерировать пароли.

Для того, что бы его запустить, необходимо через **BotFather** в телеграме создать бота и полученный токен возвращать из метода `getBotToken` в классе `TelegramBot`

У приложения две точки входа: `Main#main` и `TelegramBot#main`. Первая используется для запуска генератора паролей в консольном режиме. Вторая, в режиме телеграм бот.
